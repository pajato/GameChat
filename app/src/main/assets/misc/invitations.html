How to Send Invitations

The Floating Action Button (FAB) allows you to send invitations to friends via the "Invite friends" menu item. There are two modes:

  • When showing the available groups, you can select from these groups and their rooms,
  • When showing the rooms in a group, you can select which rooms to invite friends to join.

The toolbar overflow menu allows you to invite friends to a room in a group:

  • When showing messages, or
  • When showing the games in a room, or
  • When playing a game.

The app menu allows you to invite friends into selected groups and rooms.
